Airobot: Multi-Agent Robotic Arm Collaboration System Based on Edge-Cloud Coordination and Federated Learning

Ziang Luo1, Aiwei Li1, Shuhan Miao2
TsingHua University1, Peking University2
Code

Airobot integrates edge-cloud collaboration and federated learning，ensuring both high performance and compliance with privacy constraints. By combining the cloud's task planning capabilities with the edge's lightweight model execution, it forms an intelligent, high-performance, and scalable framework.
